## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/osama0561/ai_clone_studio.git
cd ai_clone_studio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Open http://localhost:3000

## Deployment

This app is designed to be deployed on Vercel:

1. Push to GitHub
2. Import project in Vercel
3. Add `GEMINI_API_KEY` in Environment Variables (optional, users can enter their own)
4. Deploy!

## Usage

1. Enter your Gemini API key
2. Upload a clear photo of your face
3. Click "Generate Face Variations"
4. Download generated images
